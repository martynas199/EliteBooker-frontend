# Schema Templates

## Core SaaS Pages
Use:
- `Organization`
- `SoftwareApplication`
- `BreadcrumbList`
- `FAQPage` (only where FAQ content is visible)

## Industry Pages
Use:
- `Service`
- `FAQPage`
- `BreadcrumbList`

## Location + Industry Pages
Use:
- `Service`
- `SoftwareApplication`
- `FAQPage`
- `BreadcrumbList`

## Blog / Guides
Use:
- `Article`
- `BreadcrumbList`
- `HowTo` only for explicit step-by-step instructional pages

## Validation Rules
- Every schema object must match visible on-page content.
- No fabricated ratings/review counts.
- JSON-LD must be valid and parseable with Google Rich Results Test.
