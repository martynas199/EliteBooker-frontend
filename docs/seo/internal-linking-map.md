# Internal Linking Map

## Hubs

- `/`
- `/features`
- `/pricing`
- `/compare`
- `/solutions`

## Hub -> Spokes

- `/features` links to all feature detail pages.
- `/compare` links to competitor comparison pages.
- `/solutions` links to city+niche local solution pages.

## Cross-Cluster Links

- Each local solution page links to:
  - nearby-city same-niche pages
  - same-city related-niche pages
- Industry pages should link to top city pages for the same industry.
